04 / HTTP API

Mock processor contract.

Single endpoint today; swap the handler for OpenAI, Anthropic, or your own worker.

POST /api/ai/process

JSON body:

{
  "text": "string (required)",
  "mode": "summarize | translate | tone | explain",
  "target_lang": "optional for translate",
  "target_tone": "optional for tone"
}

Returns { "success": true, "result": "…" } or an error object.

Rate & keys

  • Demo adds ~1s artificial delay.
  • No API key required for the mock pipeline.
  • Replace ai_process() in Flask when you integrate a provider.

Related routes

  • Studio UI — calls this endpoint from the browser.
  • Presets — client-only; could POST the same payload after loading a preset.
